# Perubahan 2026-06-07 — Naikkan maxTvl 280K → 400K

## Latar Belakang

Copy engine auto-deploy berhasil diimplementasikan dan berfungsi. Namun, 2 dari 6 kandidat COPY signal di-block oleh safety gate `maxTvl`:

- **ZEC-USDC**: TVL $535,607 > maxTvl $280,000 → BLOCKED
- **CARDS-USDC**: TVL $350,940 > maxTvl $280,000 → BLOCKED

## Analisa

| Parameter | Default (config.js) | User Set (sebelum) | Baru |
|-----------|-------------------|--------------------|------|
| `maxTvl` | 150,000 | 280,000 | **400,000** |
| `deployAmountSol` | 0.5 | 0.13 | **0.13 (tetap)** |

**Alasan kuat untuk naikkan maxTvl:**
1. User sudah pernah naikkan dari 150K → 280K — ingin akses pool lebih besar
2. Pool $350K-$535K lebih likuid dan stabil dibanding pool kecil
3. Decision engine sudah filter fee_tvl_ratio, organic, volatility, confidence
4. `deployAmountSol` tidak diubah — posisi tetap kecil (0.13 SOL)

## Perubahan

### File: `user-config.json`

```diff
- "maxTvl": 280000,
+ "maxTvl": 400000,
```
